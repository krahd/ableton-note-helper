## Primary one-time post: Ableton Note forum

### Title

Free interactive Note and Push grid helper for scales, chords and voicings

### Body

I wanted accurate practice diagrams for Ableton Note, but static charts became crowded and often obscured the actual grid geometry. I built a free interactive reference that generates the iPad, iPhone and Push layouts algorithmically.

It supports multiple simultaneous scales, modes, chords, arpeggios and exact voicings. You can also tap pads to identify chords, alternative interpretations and inversions. The app runs entirely in the browser, requires no account, performs no tracking and is open source.

Live app: https://krahd.github.io/ableton-note-helper/

## Optional broader one-time post: Show HN

### Title

Show HN: Interactive music-theory reference for Ableton Note and Push grids

### First comment

I built this because I wanted an accurate way to compare scales, chords and exact voicings on Ableton Note's different iPad and iPhone grids, as well as the Push layout.

The grid geometry, patterns and voicings are generated rather than stored as diagrams. Exact voicings illuminate one physical pad per note, while scales and chords illuminate pitch classes across the grid. The tapped-note analyser keeps octave information so that it can identify bass notes and inversions.

It is a dependency-free static application hosted on GitHub Pages. There is no account, server-side component or tracking.
